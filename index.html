🔥 הגדרת Firebase

Firebase מאפשר סנכרון בזמן אמת בין נייד, טאבלט ומחשב.
3 דקות הגדרה חד-פעמית:
1. כנס ל- console.firebase.google.com
2. "צור פרויקט" → תן שם → המשך
3. לחץ על ⚙️ Project Settings → "Your apps" → Web
4. רשום שם → "Register" → העתק את ה-firebaseConfig
5. בתפריט שמאל: Build → Realtime Database → "Create" → Start in test mode
